✊✋✌️ Stone Paper Scissors Game
A fun and interactive Stone Paper Scissors game built using HTML, CSS, and JavaScript. Play against the computer and test your luck and logic!

🚀 Features :
🤖 Play against computer (random moves)
🎯 Instant result display (Win / Lose / Draw)
📊 Score tracking (optional if you implemented)
🔄 Reset / Play again functionality
💡 Simple and clean user interface


🛠️ Tech Stack
HTML – Structure of the game
CSS – Styling and layout
JavaScript – Game logic and randomness

🧠 Game Logic
The player selects one of:
 * Stone
 * Paper
 * Scissors
The computer randomly generates its choice.
Rules:
-> Stone beats Scissors
-> Scissors beats Paper
-> Paper beats Stone
-> Same choices → Draw
